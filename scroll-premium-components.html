01 / Clip-Path Reveal

Cortinas de
Texto Fluidas

Cada linha de texto sobe suavemente de baixo para cima como uma cortina de teatro. O efeito usa overflow: hidden + translateY para revelações cinematográficas precisas a cada scroll.

02 / Parallax Assimétrico

Três Imagens,
Três Velocidades

Cada coluna de imagem se move em velocidade ligeiramente diferente, criando percepção de profundidade tridimensional real durante o scroll. Puro transform no eixo Y.

03 / Contadores Animados

Números que
Ganham Vida

Os números contam do zero ao valor final no momento exato em que entram na viewport. Implementado com IntersectionObserver nativo para performance máxima sem bibliotecas.

04 / Horizontal Pin Scroll

Narrativa em
Trilho Lateral

A seção que você está lendo agora pina a tela enquanto o conteúdo desliza horizontalmente, criando uma narrativa cinematográfica fluída. Calculado em tempo real via scroll listener.
